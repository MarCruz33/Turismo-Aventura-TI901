# Turismo Aventura - Sitio Web

## Descripción
Sitio web de turismo desarrollado como práctica colaborativa para la asignatura de Desarrollo Web Integral. El sitio incluye funcionalidades como autenticación con Gmail, consumo de APIs (JSONPlaceholder, clima simulado, reservas de vuelos) y una interfaz atractiva para promover destinos turísticos.

## APIs Consumidas
- **JSONPlaceholder** (Usuarios) - Para mostrar una lista de usuarios de prueba.
- **Clima (simulado)** - Para mostrar el clima de una ciudad ingresada por el usuario.
- **Reservas y Vuelos** - Simulación de una API que muestra vuelos disponibles y permite realizar reservas.
